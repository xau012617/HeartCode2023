About

    We are a Secondary 3 duo participating in the Heartcode Programme. Through this programme, we created this website: Flame-proofing The Internet to raise awareness about cyber-bullying, especially flaming.
    So why did we choose the theme Flaming? Well, we felt that online flaming is quite prevalent among youths recently due to the rapid advancement of technology, as well as the COVID-19 pandemic which caused many youths to stay at home. Hence, this website was created to educate youths on how to spot and prevent online flaming so as to create a better cyberspace for all.

    The heartcode programme has enabled us to learn how to code using HTML, CSS, Bootstrap and Github within a week .With the help of our mentors,we managed to learn how to make this website.

    Success
